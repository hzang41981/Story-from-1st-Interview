用stack：
```python3
class Solution:
    def lengthOfLIS(self, nums: List[int]) -> int:
        stack = []
        for i in range(len(nums)):
            if not stack or nums[i] > stack[-1]:
                stack.append(nums[i])
            else:
                pter = len(stack) - 2
                while pter > -1 and nums[i] <= stack[pter]:
                    pter -= 1
                stack[pter + 1] = nums[i]
        return len(stack)
```
